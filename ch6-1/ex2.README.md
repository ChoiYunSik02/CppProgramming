```cpp
class Point3D {
```
> `Point3D` 클래스 선언 시작. 3차원 공간의 한 점을 표현하는 클래스다.
 
```cpp
private:
```
> 이하 멤버 변수들은 클래스 외부에서 직접 접근 불가 — **캡슐화** 원칙.
 
```cpp
    int x, y, z;
```
> 3차원 좌표를 저장하는 멤버 변수 `x`, `y`, `z`를 한 줄에 선언한다.
> 같은 타입(`int`)이면 `,`로 구분하여 한 줄 선언이 가능하다.
 
```cpp
public:
```
> 이하 생성자 및 `show()` 함수는 클래스 외부에서 호출 가능.
 
```cpp
    Point3D() { x = 0; y = 0; z = 0; }
```
> **[생성자 1]** 매개변수 없는 기본 생성자.
> `Point3D p0;` 처럼 인자 없이 객체를 생성하면 호출된다.
> x, y, z 모두 0으로 초기화한다.
 
```cpp
    Point3D(int a) { x = a; y = 0; z = 0; }
```
> **[생성자 2]** 정수 1개를 받는 생성자.
> `Point3D p1(1);` 처럼 인자 1개로 객체를 생성하면 호출된다.
> x는 `a`로, y와 z는 0으로 초기화한다.
 
```cpp
    Point3D(int a, int b) { x = a; y = b; z = 0; }
```
> **[생성자 3]** 정수 2개를 받는 생성자.
> `Point3D p2(1, 2);` 처럼 인자 2개로 객체를 생성하면 호출된다.
> x는 `a`, y는 `b`로, z는 0으로 초기화한다.
 
```cpp
    Point3D(int a, int b, int c) { x = a; y = b; z = c; }
```
> **[생성자 4]** 정수 3개를 받는 생성자.
> `Point3D p3(1, 2, 3);` 처럼 인자 3개로 객체를 생성하면 호출된다.
> x는 `a`, y는 `b`, z는 `c`로 초기화한다.
 
```cpp
    void show() {
```
> 멤버 함수 `show()` 정의 시작. 클래스 **내부**에 바로 정의하므로 **인라인 함수**로 처리된다.
 
```cpp
        cout << "3차원 점의 좌표는 " << "(" << x << "," << y << "," << z << ")" << endl;
```
> x, y, z 값을 `(x,y,z)` 형식으로 출력한다.
> `"("`, `","`, `")"` 는 문자열 리터럴로 출력 형식을 맞추기 위해 사용한다.
 
```cpp
    }
```
> `show()` 함수 끝.
 
```cpp
};
```
> `Point3D` 클래스 선언 끝. 반드시 `;` 을 붙여야 한다.
 
---
 
```cpp
int main() {
```
> 프로그램 실행의 시작점.
 
```cpp
    Point3D p0;
```
> **생성자 1** 호출 — 인자가 없으므로 `Point3D()`가 실행된다.
> `x=0`, `y=0`, `z=0`으로 초기화된다.
 
```cpp
    p0.show();
```
> `p0` 객체의 `show()`를 호출한다. → `3차원 점의 좌표는 (0,0,0)` 출력.
 
```cpp
    Point3D p1(1);
```
> **생성자 2** 호출 — 인자가 1개(1)이므로 `Point3D(int a)`가 실행된다.
> `x=1`, `y=0`, `z=0`으로 초기화된다.
 
```cpp
    p1.show();
```
> `p1` 객체의 `show()`를 호출한다. → `3차원 점의 좌표는 (1,0,0)` 출력.
 
```cpp
    Point3D p2(1, 2);
```
> **생성자 3** 호출 — 인자가 2개(1, 2)이므로 `Point3D(int a, int b)`가 실행된다.
> `x=1`, `y=2`, `z=0`으로 초기화된다.
 
```cpp
    p2.show();
```
> `p2` 객체의 `show()`를 호출한다. → `3차원 점의 좌표는 (1,2,0)` 출력.
 
```cpp
    Point3D p3(1, 2, 3);
```
> **생성자 4** 호출 — 인자가 3개(1, 2, 3)이므로 `Point3D(int a, int b, int c)`가 실행된다.
> `x=1`, `y=2`, `z=3`으로 초기화된다.
 
```cpp
    p3.show();
```
> `p3` 객체의 `show()`를 호출한다. → `3차원 점의 좌표는 (1,2,3)` 출력.
 
```cpp
    return 0;
```
> `main()` 함수가 정상 종료됨을 운영체제에 알린다.
 
```cpp
}
```
> `main()` 함수 끝.

## 📝 핵심 정리
 
| 생성자 | 호출 형태 | x | y | z |
|--------|-----------|---|---|---|
| `Point3D()` | `Point3D p0;` | 0 | 0 | 0 |
| `Point3D(int a)` | `Point3D p1(1);` | 1 | 0 | 0 |
| `Point3D(int a, int b)` | `Point3D p2(1, 2);` | 1 | 2 | 0 |
| `Point3D(int a, int b, int c)` | `Point3D p3(1, 2, 3);` | 1 | 2 | 3 |
 
- 매개변수 개수가 0, 1, 2, 3개로 모두 달라 4개의 생성자가 각각 구별된다.
- 지정되지 않은 좌표는 항상 `0`으로 초기화 — 일관된 기본값 정책.
- `show()`를 클래스 내부에 직접 정의하면 **인라인 함수**로 처리되어 함수 호출 오버헤드가 줄어든다.
