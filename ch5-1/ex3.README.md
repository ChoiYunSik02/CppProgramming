## 📋 문제 요약

함수 호출문 `res = GetLatterString(&s1, &s2)` 를 참고하여  
`GetLatterString` 함수의 **선언과 정의를 추가**하시오.  
- 주소에 의한 호출 사용
- 매개변수는 `string*` 객체 포인터로 선언
- 리턴값은 `string` 객체로 선언

---

```cpp
string GetLatterString(string* s1, string* s2)
{
```
> **함수 정의**  
> - 매개변수 `string* s1` : `s1` 객체의 주소를 저장하는 포인터 변수  
> - 매개변수 `string* s2` : `s2` 객체의 주소를 저장하는 포인터 변수  
> - 포인터는 객체가 아니므로 **생성자·소멸자가 호출되지 않음**  
> - 주소값(4~8바이트)만 복사되므로 **메모리·속도 효율이 높음**  
> - 리턴 타입 `string` : 함수 종료 시 `string` 객체를 값으로 반환

---

```cpp
    if (*s1 > *s2)
```
> **역참조 연산자 `*` 로 포인터가 가리키는 실제 객체에 접근**  
> - `s1` : `s1` 객체의 **주소**  
> - `*s1` : 그 주소에 저장된 **실제 `string` 객체** (`"hello"`)  
> - `*s2` : 그 주소에 저장된 **실제 `string` 객체** (`"world"`)  
> - `string` 의 `>` 연산자로 사전 순(lexicographic) 비교  
> - `'h'(104) < 'w'(119)` 이므로 `*s1 > *s2` 는 **거짓**

---

```cpp
        return *s1;
```
> `*s1` 이 `*s2` 보다 사전 순으로 뒤에 올 경우 `*s1` (역참조한 실제 객체)을 반환  
> - 반환 시 컴파일러가 **임시 객체**를 자동 생성하여 값을 저장  
> - 포인터 `s1` 이 아닌 `*s1` (객체 자체)을 반환한다는 점에 주의

---

```cpp
    else
        return *s2;
```
> `*s1 <= *s2` 인 경우 `*s2` (역참조한 실제 객체)를 반환  
> - `"hello" < "world"` 이므로 이 분기가 실행되어 `"world"` 가 반환됨  
> - `*s1 == *s2` 인 경우에도 `*s2` 가 반환됨

---

```cpp
}
```
> `GetLatterString()` 함수 종료  
> - 포인터 매개변수 `s1`, `s2` 는 스택에서 소멸  
> - 단, 포인터가 가리키던 **원본 객체(`main()`의 `s1`, `s2`)는 소멸되지 않음**

## 📊 함수 호출 흐름 정리

```
main() 스택                         GetLatterString() 스택
──────────────────────────────────────────────────────────
s1 │ "hello" │ 주소: 0x100          s1 │ 0x100 │ ← 주소만 복사
s2 │ "world" │ 주소: 0x120          s2 │ 0x120 │ ← 주소만 복사

                    &s1(0x100) ──▶
                    &s2(0x120) ──▶
                                    *s1 → "hello" (역참조)
                                    *s2 → "world" (역참조)

                                    "hello" > "world" → false
                                    return *s2 → "world"

                    ◀── 임시객체 ── "world"
res = "world"
```

## 📌 핵심 정리

| 항목 | 내용 |
|------|------|
| 호출 방식 | **주소에 의한 호출 (call by address)** |
| 전달 내용 | `&s1`, `&s2` — 객체의 **메모리 주소** |
| 역참조 | `*s1`, `*s2` — 포인터가 가리키는 **실제 객체**에 접근 |
| 복사 발생 | ❌ 객체 복사 없음, 포인터 값만 복사 |
| 비교 연산 | `string` 의 `>` 연산자 — 사전 순(lexicographic) 비교 |
| 반환 방식 | `*s1` 또는 `*s2` — 역참조한 실제 객체를 임시 객체로 반환 |
