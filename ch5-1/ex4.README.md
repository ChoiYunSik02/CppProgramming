# Chapter 05-1 실습과제4 소스코드 설명

> **주소에 의한 호출로 두 string 객체의 문자열 교환**  
> 학번: 2101091 최윤식

---

## 📋 문제 요약

다음 결과가 나오도록 두 개의 `string` 객체의 문자열을 교환하는  
`SwapString` 함수를 만드시오.

```
호출전 s1:hello s2:world
호출후 s1:world s2:hello
```

---

> - `using namespace std` : `std::` 접두사 없이 표준 라이브러리 사용

> **함수 정의**  
> - 매개변수 `string* s1` : `main()` 의 `s1` 객체 주소를 저장하는 포인터  
> - 매개변수 `string* s2` : `main()` 의 `s2` 객체 주소를 저장하는 포인터  
> - 포인터는 객체가 아니므로 **생성자·소멸자 호출 없음**

```cpp

```
> **임시 객체 선언**  
> - 두 값을 교환할 때 한쪽 값이 덮어써지기 전에 **백업용으로 사용**  
> - `string` 기본 생성자가 호출되어 빈 문자열 `""` 로 초기화됨

> - `*s1` : 포인터 역참조 → `main()` 의 `s1` 원본 객체 (`"hello"`) 에 접근  
> - `tmp` 에 `"hello"` 가 저장됨  
> - 이 시점의 상태 : `tmp="hello"`, `*s1="hello"`, `*s2="world"`

---

```cpp

```

> - `main()` 의 원본 `s1` 이 `"world"` 로 변경됨  
> - 이 시점의 상태 : `tmp="hello"`, `*s1="world"`, `*s2="world"`

---

```cpp

```

> - `main()` 의 원본 `s2` 가 `"hello"` 로 변경됨  
> - 이 시점의 상태 : `tmp="hello"`, `*s1="world"`, `*s2="hello"` ✅ 교환 완료

## 📊 교환 과정 단계별 상태표

| 단계 | 코드 | tmp | *s1 (원본 s1) | *s2 (원본 s2) |
|------|------|-----|--------------|--------------|
| 초기 상태 | — | `""` | `"hello"` | `"world"` |
| ① 백업 | `tmp = *s1` | `"hello"` | `"hello"` | `"world"` |
| ② 덮어쓰기 | `*s1 = *s2` | `"hello"` | `"world"` | `"world"` |
| ③ 복원 | `*s2 = tmp` | `"hello"` | `"world"` | `"hello"` |

## 📊 함수 호출 흐름 정리

```
main() 스택                          SwapString() 스택
──────────────────────────────────────────────────────
s1 │ "hello" │ 주소: 0x100           s1 │ 0x100 │ ← 주소만 복사
s2 │ "world" │ 주소: 0x120           s2 │ 0x120 │ ← 주소만 복사

                    &s1(0x100) ──▶
                    &s2(0x120) ──▶
                                     tmp = *s1  →  tmp = "hello"
                                     *s1 = *s2  →  s1 = "world"  (원본 변경)
                                     *s2 = tmp  →  s2 = "hello"  (원본 변경)
                                     return tmp → "hello"

함수 종료 후:
s1 │ "world" │  ← 원본이 직접 변경됨
s2 │ "hello" │  ← 원본이 직접 변경됨
```

## ❌ 값에 의한 호출로 하면 왜 안 될까?

```cpp
// 잘못된 예시 — 값에 의한 호출
string SwapString(string s1, string s2) {
    string tmp = s1;
    s1 = s2;          // 복사본 s1만 바뀜
    s2 = tmp;         // 복사본 s2만 바뀜
    return tmp;
    // main()의 원본 s1, s2는 전혀 변하지 않음!
}
```

> 값에 의한 호출은 **복사본끼리 교환**하는 것이므로  
> 함수가 종료되면 복사본은 소멸하고 원본은 그대로 남는다.  
> 원본을 바꾸려면 반드시 **주소에 의한 호출**을 사용해야 한다.

## 📌 핵심 정리

| 항목 | 내용 |
|------|------|
| 호출 방식 | **주소에 의한 호출 (call by address)** |
| 전달 내용 | `&s1`, `&s2` — 객체의 **메모리 주소** |
| 역참조 | `*s1`, `*s2` — 포인터가 가리키는 **원본 객체** 직접 수정 |
| 교환 방법 | `tmp` 임시 객체로 백업 후 **3단계 대입** |
| 원본 변경 여부 | ✅ `main()` 의 `s1`, `s2` 가 직접 교환됨 |
| 리턴값 | 교환 전 `*s1` 의 원래 값 (`tmp`) 반환 — `main()` 에서는 미사용 |
