## 헤더 및 네임스페이스

```cpp
#include <iostream>
using namespace std;
```

- `#include <iostream>` : `cin`, `cout` 사용을 위한 헤더 파일
- `using namespace std` : `std::` 접두사 없이 사용 가능하게 설정

## 클래스 선언

```cpp
class Sphere {
public:
    int radius;
    double getVolume();
    Sphere();
    Sphere(int r);
};
```

| 멤버 | 종류 | 설명 |
|---|---|---|
| `radius` | 멤버변수 | 구의 반지름 |
| `getVolume()` | 멤버함수 | 구의 부피를 계산하여 반환 |
| `Sphere()` | 위임 생성자 | 인자 없이 객체 생성 시 호출, 타겟 생성자에 위임 |
| `Sphere(int r)` | 타겟 생성자 | 실제 초기화 작업을 담당 |

- `public` : 클래스 외부에서 모든 멤버에 자유롭게 접근 가능
- 반환형이 `double`인 이유 : 부피 공식에서 `4.0/3.0`을 곱하므로 소수점이 발생할 수 있기 때문

---

## 멤버함수 구현

### getVolume() — 부피 계산

```cpp
double Sphere::getVolume() {
    return (4.0 / 3.0) * 3.14 * radius * radius * radius;
}
```

- 구의 부피 공식 : **4/3 × π × 반지름³**
- `3.14`를 π(파이) 근삿값으로 사용
- `4.0/3.0` 으로 작성한 이유 : `4/3`으로 쓰면 정수 나눗셈이 되어 결과가 `1`이 되기 때문에, 실수 나눗셈이 되도록 `4.0/3.0`으로 작성
- `radius * radius * radius` : 반지름의 세제곱 (radius³)

---

### Sphere() — 위임 생성자

```cpp
Sphere::Sphere() : Sphere(1) { }
```

- 인자 없이 객체를 생성할 때 호출되는 생성자
- 직접 초기화하지 않고 `Sphere(1)` 즉, **타겟 생성자에게 초기화를 위임**
- `radius = 1` 로 초기화됨
- 몸체 `{ }` 가 비어있는 이유 : 실제 초기화는 타겟 생성자가 담당하기 때문

---

### Sphere(int r) — 타겟 생성자

```cpp
Sphere::Sphere(int r) : radius(r) { }
```

- 정수 하나를 인자로 받아 직접 멤버변수를 초기화하는 생성자
- **멤버 초기화 리스트** `: radius(r)` 를 사용하여 객체 생성과 동시에 초기화
- 위임 생성자와 일반 호출 모두 이 생성자를 통해 최종 초기화가 이루어짐
- 몸체 `{ }` 가 비어있는 이유 : 초기화 리스트에서 모든 초기화가 완료되기 때문

## main() 함수

```cpp
int main() {
    Sphere sph1;
    cout << "구의 부피는 " << sph1.getVolume() << endl;

    Sphere sph2(3);
    cout << "구의 부피는 " << sph2.getVolume() << endl;

    return 0;
}
```

| 코드 | 설명 |
|---|---|
| `Sphere sph1` | 인자 없이 객체 생성 → 위임 생성자 호출 → radius=1 |
| `sph1.getVolume()` | 4/3 × 3.14 × 1³ = **4.19** 출력 |
| `Sphere sph2(3)` | 인자를 주어 객체 생성 → 타겟 생성자 호출 → radius=3 |
| `sph2.getVolume()` | 4/3 × 3.14 × 3³ = **113.04** 출력 |

---

## 생성자 호출 흐름

```
sph1 생성
  └→ Sphere()      (위임 생성자 호출)
       └→ Sphere(1) (타겟 생성자로 위임 → radius=1)

sph2 생성
  └→ Sphere(3)     (타겟 생성자 직접 호출 → radius=3)
```

## 핵심 포인트 정리

| 개념 | 설명 |
|---|---|
| 위임 생성자 | 초기화를 직접 하지 않고 타겟 생성자에게 넘김 |
| 타겟 생성자 | 실제 멤버변수 초기화를 담당하는 생성자 |
| 멤버 초기화 리스트 | 생성자 몸체 대신 `: radius(r)` 형식으로 초기화 |
| `4.0/3.0` | 정수 나눗셈 오류를 방지하기 위한 실수형 표현 |
