```cpp
class Point3D {
```
> `Point3D` 클래스 선언 시작. 3차원 공간의 한 점을 표현하는 클래스다.

```cpp
private:
```
> 이하 멤버 변수들은 클래스 외부에서 직접 접근 불가 — **캡슐화** 원칙.

```cpp
    int x, y, z;
```
> 3차원 좌표를 저장하는 멤버 변수 `x`, `y`, `z`를 한 줄에 선언한다.
> 같은 타입(`int`)이면 `,` 로 구분하여 한 줄 선언이 가능하다.

```cpp
public:
```
> 이하 생성자와 `show()` 함수는 클래스 외부에서 호출 가능.

```cpp
    Point3D(int a = 0, int b = 0, int c = 0) { x = a; y = b; z = c; }
```
> **생성자 1개** — 세 매개변수 모두 디폴트값(0)을 가지므로
> 인자를 0개, 1개, 2개, 3개 어떻게 넘겨도 이 하나의 생성자로 처리된다.
>
> | 호출 형태 | a | b | c | 설명 |
> |-----------|---|---|---|------|
> | `Point3D p0;` | 0 | 0 | 0 | 모두 디폴트값 사용 |
> | `Point3D p1(1);` | 1 | 0 | 0 | b, c는 디폴트값 사용 |
> | `Point3D p2(1, 2);` | 1 | 2 | 0 | c만 디폴트값 사용 |
> | `Point3D p3(1, 2, 3);` | 1 | 2 | 3 | 모두 직접 전달 |
>
> 6장-1 실습과제4에서는 생성자를 4개 만들었지만,
> 디폴트 매개변수를 사용하면 **생성자 1개**로 동일한 기능을 구현할 수 있다.

```cpp
    void show() {
```
> 멤버 함수 `show()` 정의 시작. 클래스 **내부**에 바로 정의하므로 **인라인 함수**로 처리된다.

```cpp
        cout << "3차원 점의 좌표는 " << "(" << x << "," << y << "," << z << ")" << endl;
```
> x, y, z 값을 `(x,y,z)` 형식으로 출력한다.
> `"("`, `","`, `")"` 는 출력 형식을 맞추기 위한 문자열 리터럴이다.

```cpp
    }
```
> `show()` 함수 끝.

```cpp
};
```
> `Point3D` 클래스 선언 끝. 반드시 `;` 을 붙여야 한다.

---

```cpp
int main() {
```
> 프로그램 실행의 시작점.

```cpp
    Point3D p0;
```
> 인자 없이 객체 생성 → `a=0(디폴트)`, `b=0(디폴트)`, `c=0(디폴트)`으로 초기화.
> 내부적으로 `Point3D(0, 0, 0)` 과 동일하게 처리된다.

```cpp
    p0.show();
```
> `p0` 객체의 `show()`를 호출한다. → `3차원 점의 좌표는 (0,0,0)` 출력.

```cpp
    Point3D p1(1);
```
> 인자 1개(1)로 객체 생성 → `a=1`, `b=0(디폴트)`, `c=0(디폴트)`으로 초기화.
> 내부적으로 `Point3D(1, 0, 0)` 과 동일하게 처리된다.

```cpp
    p1.show();
```
> `p1` 객체의 `show()`를 호출한다. → `3차원 점의 좌표는 (1,0,0)` 출력.

```cpp
    Point3D p2(1, 2);
```
> 인자 2개(1, 2)로 객체 생성 → `a=1`, `b=2`, `c=0(디폴트)`으로 초기화.
> 내부적으로 `Point3D(1, 2, 0)` 과 동일하게 처리된다.

```cpp
    p2.show();
```
> `p2` 객체의 `show()`를 호출한다. → `3차원 점의 좌표는 (1,2,0)` 출력.

```cpp
    Point3D p3(1, 2, 3);
```
> 인자 3개(1, 2, 3)로 객체 생성 → `a=1`, `b=2`, `c=3`으로 초기화.
> 디폴트값 없이 세 인자가 모두 직접 전달된다.

```cpp
    p3.show();
```
> `p3` 객체의 `show()`를 호출한다. → `3차원 점의 좌표는 (1,2,3)` 출력.

```cpp
    return 0;
```
> `main()` 함수가 정상 종료됨을 운영체제에 알린다.

```cpp
}
```
> `main()` 함수 끝.

## 📝 핵심 정리

| 호출 형태 | 실제 처리 | x | y | z |
|-----------|-----------|---|---|---|
| `Point3D p0;` | `Point3D(0, 0, 0)` | 0 | 0 | 0 |
| `Point3D p1(1);` | `Point3D(1, 0, 0)` | 1 | 0 | 0 |
| `Point3D p2(1, 2);` | `Point3D(1, 2, 0)` | 1 | 2 | 0 |
| `Point3D p3(1, 2, 3);` | `Point3D(1, 2, 3)` | 1 | 2 | 3 |

- **디폴트 매개변수**를 사용하면 6장-1의 생성자 4개를 **생성자 1개**로 줄일 수 있다.
- 디폴트값은 반드시 **뒤쪽 매개변수부터** 지정해야 한다.
  - `(int a = 0, int b, int c = 0)` → ❌ 컴파일 오류 (중간에 일반 매개변수 불가)
  - `(int a = 0, int b = 0, int c = 0)` → ✅ 정상
- `show()`를 클래스 내부에 직접 정의하면 **인라인 함수**로 처리되어 함수 호출 오버헤드가 줄어든다.
