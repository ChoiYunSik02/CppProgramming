```cpp
class Rectangle {
```
> `Rectangle` 클래스 선언 시작.

```cpp
private:
```
> 이하 멤버 변수들은 클래스 외부에서 직접 접근 불가 — **캡슐화** 원칙.

```cpp
    int width;
```
> 사각형의 **폭**을 저장하는 멤버 변수.

```cpp
    int height;
```
> 사각형의 **높이**를 저장하는 멤버 변수.

```cpp
public:
```
> 이하 생성자와 `show()` 함수는 클래스 외부에서 호출 가능.

```cpp
    Rectangle(int w = 1, int h = 1) { width = w; height = h; }
```
> **생성자 1개** — 디폴트 매개변수를 사용하여 3가지 호출 형태를 모두 처리한다.
>
> | 호출 형태 | w | h | 설명 |
> |-----------|---|---|------|
> | `Rectangle rect0;` | 1 | 1 | 둘 다 디폴트값 사용 |
> | `Rectangle rect1(10);` | 10 | 1 | h만 디폴트값 사용 |
> | `Rectangle rect2(10, 20);` | 10 | 20 | 둘 다 직접 전달 |
>
> 6장-1 실습과제3에서는 생성자를 3개 만들었지만,
> 디폴트 매개변수를 사용하면 **생성자 1개**로 동일한 기능을 구현할 수 있다.

```cpp
    void show();
```
> 멤버 함수 `show()`의 **선언**만 클래스 내부에 작성한다.

```cpp
};
```
> `Rectangle` 클래스 선언 끝. 반드시 `;` 을 붙여야 한다.

```cpp
void Rectangle::show() {
```

> 클래스 **외부**에서 `show()` 함수를 정의한다.
> `Rectangle::` 은 이 함수가 `Rectangle` 클래스의 멤버임을 나타내는 **스코프 연산자(::)** 다.

```cpp
    cout << "사각형 폭은 " << width << " 높이는 " << height << endl;
```

> 멤버 변수 `width`와 `height` 값을 출력한다.
> `private` 멤버이지만 멤버 함수 내부이므로 접근 가능하다.

```cpp
}
```
> `show()` 함수 끝.

```cpp
int main() {
```
> 프로그램 실행의 시작점.

```cpp
    Rectangle rect0;
```
> 인자 없이 객체 생성 → `w=1(디폴트)`, `h=1(디폴트)`로 초기화.
> 내부적으로 `Rectangle(1, 1)` 과 동일하게 처리된다.

```cpp
    rect0.show();
```
> `rect0` 객체의 `show()`를 호출한다. → `사각형 폭은 1 높이는 1` 출력.

```cpp
    Rectangle rect1(10);
```
> 인자 1개(10)로 객체 생성 → `w=10`, `h=1(디폴트)`로 초기화.
> 내부적으로 `Rectangle(10, 1)` 과 동일하게 처리된다.

```cpp
    rect1.show();
```
> `rect1` 객체의 `show()`를 호출한다. → `사각형 폭은 10 높이는 1` 출력.

```cpp
    Rectangle rect2(10, 20);
```
> 인자 2개(10, 20)로 객체 생성 → `w=10`, `h=20`으로 초기화.
> 디폴트값 없이 두 인자가 모두 직접 전달된다.

```cpp
    rect2.show();
```
> `rect2` 객체의 `show()`를 호출한다. → `사각형 폭은 10 높이는 20` 출력.

```cpp
    return 0;
```
> `main()` 함수가 정상 종료됨을 운영체제에 알린다.

```cpp
}
```
> `main()` 함수 끝.

## 📝 핵심 정리

| 호출 형태 | 실제 처리 | width | height |
|-----------|-----------|-------|--------|
| `Rectangle rect0;` | `Rectangle(1, 1)` | 1 | 1 |
| `Rectangle rect1(10);` | `Rectangle(10, 1)` | 10 | 1 |
| `Rectangle rect2(10, 20);` | `Rectangle(10, 20)` | 10 | 20 |

- **디폴트 매개변수**를 사용하면 6장-1의 생성자 3개를 **생성자 1개**로 줄일 수 있다.
- 디폴트값은 **뒤쪽 매개변수부터** 지정해야 한다 — `(int w = 1, int h)` 는 컴파일 오류.
- 디폴트 매개변수와 중복 함수(오버로딩)를 **동시에 사용하면 모호성 오류**가 발생할 수 있다.
