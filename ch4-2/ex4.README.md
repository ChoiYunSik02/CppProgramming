# 실습과제4 - 실수 배열 동적 할당 및 최대값 계산

> 참고: 명품 C++ 프로그래밍 4장 (객체 포인터와 객체 배열, 객체의 동적 생성) - 예제 4-6 유사

---

## 1. 과제 설명

저장할 실수의 개수를 먼저 입력 받아 **동적 할당(new)** 을 이용하여 메모리를 할당하고,  
n개의 실수를 입력 받아 **최대값**을 계산하여 출력하는 프로그램이다.

---

## 2. 코드 흐름 (단계별 설명)

### Step 1. 실수 개수 입력 및 유효성 검사

```cpp
double n;
cin >> n;
if (n <= 0) return -1;
```

- 배열의 크기로 사용할 값 `n`을 `double` 타입으로 입력 받는다.
- `n`이 0 이하이면 의미 없는 입력이므로 `-1`을 반환하고 즉시 종료한다.

> ⚠️ **주의사항**: 배열의 크기는 **정수(`int`)** 여야 한다.  
> `n`을 `double`로 선언하면 `new double[n]`에서 `double`을 배열 크기로 사용하는 것이므로  
> 컴파일러 경고가 발생할 수 있다. 실무에서는 `int n`으로 선언하는 것이 올바르다.
>
> ```cpp
> // 권장 방식
> int n;
> cin >> n;
> ```

---

### Step 2. 동적 메모리 할당

```cpp
double* p = new double[n];
if (!p) {
    cout << "메모리를 할당할 수 없습니다.";
    return -1;
}
```

- `new double[n]`으로 **힙(Heap) 영역**에 `n`개의 `double` 배열을 동적 할당한다.
- `double` 타입은 1개당 **8바이트**이므로, n=5이면 총 **40바이트**가 힙에 할당된다.
- `if (!p)` 조건으로 힙 메모리 부족 시 할당 실패(`NULL` 반환)를 검사한다.

```
스택(Stack)       힙(Heap)
┌──────┐         ┌──────┬──────┬──────┬──────┬──────┐
│  p ●─┼────────▶│ p[0] │ p[1] │ p[2] │ p[3] │ p[4] │  (n=5, 각 8바이트)
└──────┘         └──────┴──────┴──────┴──────┴──────┘
  포인터 p              동적 할당된 double 배열 (총 40바이트)
```

---

### Step 3. 배열에 실수 입력

```cpp
cout << n << "개의 실수를 입력하시오. " << endl;
for (int i = 0; i < n; i++) {
    cin >> p[i];
}
```

- 안내 메시지는 반복문 **밖**에서 한 번만 출력한다.
- `p[i]`(`*(p+i)`와 동일)로 힙 배열의 각 원소에 순서대로 접근하여 입력값을 저장한다.

| 인덱스 | 표현식 | 저장값 (예시) |
|:------:|:------:|:------------:|
| 0 | `p[0]` | 10.5 |
| 1 | `p[1]` | 20.78 |
| 2 | `p[2]` | -3.14 |
| 3 | `p[3]` | 3.14 |
| 4 | `p[4]` | -8.74 |

---

### Step 4. 최대값 계산

```cpp
double max = p[0];
for (int i = 0; i < n; i++) {
    if (p[i] > max) max = p[i];
}
```

#### 최대값 탐색 알고리즘

- **첫 번째 원소 `p[0]`을 최대값의 초기값으로 설정**한다.
- 이후 원소들을 순서대로 비교하여, 현재 원소가 `max`보다 크면 `max`를 갱신한다.
- 모든 원소를 탐색한 후 `max`에는 배열 전체의 최대값이 남는다.

#### 탐색 과정 추적 (예시 기준)

```
초기값  → max = p[0] = 10.5

i=0: p[0]=10.5  → 10.5  > 10.5  ? No  → max = 10.5  (변화 없음)
i=1: p[1]=20.78 → 20.78 > 10.5  ? Yes → max = 20.78 ✅ 갱신
i=2: p[2]=-3.14 → -3.14 > 20.78 ? No  → max = 20.78 (변화 없음)
i=3: p[3]=3.14  → 3.14  > 20.78 ? No  → max = 20.78 (변화 없음)
i=4: p[4]=-8.74 → -8.74 > 20.78 ? No  → max = 20.78 (변화 없음)

최종 결과: 최대값은 20.78
```

> 💡 **최솟값**을 구하려면 비교 부등호 방향만 반대로 바꾸면 된다.
> ```cpp
> double min = p[0];
> for (int i = 0; i < n; i++) {
>     if (p[i] < min) min = p[i];  // '>' → '<' 로 변경
> }
> ```

---

### Step 5. 동적 메모리 반환

```cpp
delete[] p;
```

- `new[]`로 할당한 배열은 반드시 **`delete[]`** 로 반환해야 한다.
- `delete`(대괄호 없음)를 사용하면 첫 번째 원소만 소멸되는 **비정상 반환**이 발생한다.
- 반환하지 않으면 **메모리 누수(Memory Leak)** 가 발생한다.

## 3. 실습과제3과의 차이점 비교

| 구분 | 실습과제3 | 실습과제4 |
|:----:|:---------:|:---------:|
| **데이터 타입** | `int` (정수) | `double` (실수) |
| **포인터 타입** | `int* p` | `double* p` |
| **원소 크기** | 4바이트 | 8바이트 |
| **계산 목표** | 평균값 (`sum / n`) | 최대값 (비교 탐색) |
| **알고리즘** | 누적 합산 후 나눗셈 | 초기값 설정 후 순차 비교 |
