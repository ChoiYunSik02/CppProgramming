# 4-2장 실습과제 3번 - 소스코드 설명

---

## 1. 과제 설명

저장할 정수의 개수를 먼저 입력 받아 **동적 할당(new)** 을 이용하여 메모리를 할당하고,  
n개의 정수를 입력 받아 **평균값**을 계산하여 출력하는 프로그램이다.

---

## 2. 코드 흐름 (단계별 설명)

### Step 1. 정수 개수 입력 및 유효성 검사

```cpp
cout << "입력할 정수의 개수를 입력하세요: ";
int n;
cin >> n;
if (n <= 0) return -1;
```

- 사용자로부터 배열의 크기로 사용할 정수 `n`을 입력 받는다.
- `n`이 0 이하이면 의미 없는 입력이므로 `-1`을 반환하고 프로그램을 즉시 종료한다.
- 이 검사가 없으면 크기가 0이거나 음수인 배열을 동적 할당하는 **비정상 동작**이 발생할 수 있다.

---

### Step 2. 동적 메모리 할당

```cpp
int* p = new int[n];
if (!p) {
    cout << "메모리를 할당할 수 없습니다.";
    return -1;
}
```

- `new int[n]`으로 **힙(Heap) 영역**에 `n`개의 정수를 저장할 배열을 동적 할당한다.
- 컴파일 시점에 크기를 고정하는 일반 배열(`int arr[4]`)과 달리,  
  **런타임(실행 중)에 사용자가 원하는 크기만큼** 메모리를 할당받을 수 있다.
- 힙 메모리가 부족하면 `new`는 `NULL(nullptr)`을 반환하므로, `if (!p)` 조건으로 할당 실패를 검사한다.

```
스택(Stack)       힙(Heap)
┌──────┐         ┌──┬──┬──┬──┬──┬──┐
│  p ●─┼────────▶│  │  │  │  │  │  │  (n=6인 경우 int 6개)
└──────┘         └──┴──┴──┴──┴──┴──┘
  포인터 p           p[0]~p[5]
```

---

### Step 3. 배열에 값 입력

```cpp
cout << n << "개의 정수를 입력하시오. " << endl;
for (int i = 0; i < n; i++) {
    cin >> p[i];
}
```

- 안내 메시지는 반복문 **밖**에서 한 번만 출력한다.
- `p[i]`는 `*(p+i)`와 동일하며, 포인터로 힙 영역의 배열 원소에 순서대로 접근하여 입력값을 저장한다.

| 인덱스 | 표현식 | 저장값 (예시) |
|:------:|:------:|:------------:|
| 0 | `p[0]` / `*(p+0)` | 10 |
| 1 | `p[1]` / `*(p+1)` | 20 |
| 2 | `p[2]` / `*(p+2)` | -3 |
| 3 | `p[3]` / `*(p+3)` | 3 |
| 4 | `p[4]` / `*(p+4)` | -10 |
| 5 | `p[5]` / `*(p+5)` | -20 |

---

### Step 4. 합산 및 평균 계산

```cpp
int sum = 0;
for (int i = 0; i < n; i++) {
    sum += p[i];
}
cout << "평균값은 " << sum / n << "입니다." << endl;
```

- `sum` 변수에 배열의 모든 원소를 누적 합산한다.
- `sum / n`으로 **정수 나눗셈(소수점 버림)** 평균값을 계산하여 출력한다.
- 예시: (10 + 20 + (-3) + 3 + (-10) + (-20)) / 6 = 0 / 6 = **0**

> ⚠️ `sum / n`은 **정수 나눗셈**이므로 소수점 이하는 버려진다.  
> 소수점 포함 평균이 필요하다면 `(double)sum / n`으로 형변환이 필요하다.

---

### Step 5. 동적 메모리 반환

```cpp
delete[] p;
```

- `new[]`로 할당한 배열은 반드시 **`delete[]`** 로 반환해야 한다.
- `delete`(대괄호 없음)를 사용하면 첫 번째 원소만 소멸되는 **비정상 반환**이 발생한다.
- 메모리를 반환하지 않으면 프로그램 종료 전까지 힙 메모리가 점유되는 **메모리 누수(Memory Leak)** 가 발생한다.

```
delete[] p 호출 후:
스택(Stack)       힙(Heap)
┌──────┐         ┌──────────────────┐
│  p ●─┼────────▶│  (해제된 영역)    │
└──────┘         └──────────────────┘
  p 포인터는        더 이상 접근 불가
  스택에 존재
```

---

## 3. 핵심 개념 요약

| 개념 | 설명 |
|:----:|:-----|
| `new int[n]` | 런타임에 n개 크기의 int 배열을 힙에 동적 할당 |
| `if (!p)` | 메모리 할당 실패(NULL 반환) 검사 |
| `p[i]` / `*(p+i)` | 포인터로 배열 원소에 접근하는 두 가지 방법 |
| `sum / n` | 정수 나눗셈으로 평균 계산 (소수점 버림) |
| `delete[] p` | 동적 할당 배열 메모리 반환 ([] 필수) |
