# 실습과제1 - 변수의 종류별 메모리 영역 정리

---

## 변수의 종류별 메모리 영역 비교

| 변수의 종류 | 메모리 영역 | 메모리 할당시점 | 메모리 해제시점 |
|:-----------:|:-----------:|:---------------:|:---------------:|
| 지역변수 | 스택(Stack) 영역 | 해당 함수(블록) 호출 시 | 해당 함수(블록) 종료 시 |
| 전역변수 | 데이터(Data) 영역 | 프로그램 시작 시 | 프로그램 종료 시 |
| 정적변수 | 데이터(Data) 영역 | 프로그램 시작 시 (또는 해당 선언부 최초 실행 시) | 프로그램 종료 시 |
| 동적메모리 | 힙(Heap) 영역 | `new` 연산자 호출 시 (런타임) | `delete` 연산자 호출 시 (런타임) |

---

## 각 항목 설명

### 지역변수 (Local Variable)
- 함수 또는 블록(`{}`) 내부에서 선언된 변수
- **스택(Stack) 영역**에 저장
- 함수가 호출될 때 할당되고, 함수가 종료되면 자동으로 소멸
- 초기화하지 않으면 쓰레기 값(garbage value)을 가짐

### 전역변수 (Global Variable)
- 함수 외부에서 선언된 변수
- **데이터(Data) 영역**에 저장
- 프로그램이 시작될 때 자동으로 할당되며, 프로그램이 종료될 때까지 유지
- 초기화하지 않으면 자동으로 0으로 초기화됨
- 프로그램의 모든 함수에서 접근 가능

### 정적변수 (Static Variable)
- `static` 키워드로 선언된 변수
- **데이터(Data) 영역**에 저장 (전역변수와 동일한 영역)
- **메모리 할당 시점**: 프로그램 시작 시 할당되나, 지역 정적변수의 경우 해당 선언부가 처음 실행될 때 초기화됨
- **메모리 해제 시점**: 프로그램이 종료될 때 소멸 (함수가 종료되어도 값이 유지됨)
- 지역 정적변수는 선언된 블록 내에서만 접근 가능하지만, 프로그램 종료까지 값이 유지되는 특성을 가짐

```cpp
void counter() {
    static int count = 0; // 최초 호출 시 한 번만 초기화
    count++;
    cout << count << endl;
}
// counter() 호출 시: 1, 2, 3, ... 순서로 누적됨
```

### 동적메모리 (Dynamic Memory)
- `new` 연산자로 런타임(실행 중)에 할당하는 메모리
- **힙(Heap) 영역**에 저장
- 프로그래머가 원하는 시점에 `new`로 할당하고, `delete`로 직접 해제해야 함
- `delete`를 호출하지 않으면 **메모리 누수(Memory Leak)** 발생

```cpp
int* p = new int;      // 힙 영역에 할당
*p = 10;
delete p;              // 반드시 직접 해제

int* arr = new int[5]; // 배열 동적 할당
delete[] arr;          // 배열 해제 시 [] 필수
```

## 메모리 구조 요약

```
+------------------+
|    코드 영역      |  ← 실행할 프로그램 코드 저장
+------------------+
|   데이터 영역     |  ← 전역변수, 정적(static)변수
+------------------+
|    힙 영역        |  ← 동적메모리 (new/delete)
|       ↓          |
|       ↑          |
|   스택 영역       |  ← 지역변수, 매개변수
+------------------+
```
