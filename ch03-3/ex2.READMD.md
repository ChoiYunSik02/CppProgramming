# 3-3장 실습과제 2번 소스코드 설명

---

## 1. 헤더 및 네임스페이스

`iostream` 헤더를 포함하여 표준 입출력 기능을 사용할 수 있도록 하고, `using namespace std;` 를 선언하여 `cout`, `endl` 등을 `std::` 접두사 없이 사용할 수 있도록 한다.

---

## 2. Triangle 클래스 선언

`Triangle` 이라는 이름의 클래스를 선언한다. 접근 지정자를 `public` 으로 설정하여 클래스 외부에서도 멤버에 자유롭게 접근할 수 있도록 한다.

### 멤버 변수

- `width` : 삼각형의 밑변을 저장하는 정수형 멤버 변수이다.
- `height` : 삼각형의 높이를 저장하는 정수형 멤버 변수이다.

### 멤버 함수 선언

- `Triangle()` : 매개변수 없이 기본 값으로 객체를 생성하는 생성자1(위임 생성자)을 선언한다.
- `Triangle(int a, int b)` : 밑변과 높이를 매개변수로 받아 객체를 생성하는 생성자2(타겟 생성자)를 선언한다.
- `~Triangle()` : 객체가 소멸될 때 자동으로 호출되는 소멸자를 선언한다.
- `getArea()` : 삼각형의 면적을 계산하여 반환하는 함수를 선언한다.

---

## 3. 생성자 구현

### 위임 생성자 - Triangle()

매개변수가 없는 기본 생성자이다. 멤버 초기화 리스트에서 `Triangle(1, 1)` 을 호출하는 방식으로 자신의 다른 생성자에게 초기화 작업을 위임한다. 이를 **위임 생성자(Delegating Constructor)** 라고 한다. 매개변수 없이 객체를 생성하면 밑변과 높이가 모두 1로 자동 설정된다.

### 타겟 생성자 - Triangle(int a, int b)

밑변(`a`)과 높이(`b`)를 매개변수로 받는 생성자이다. 멤버 초기화 리스트 `: width(a), height(b)` 를 통해 매개변수로 전달받은 값을 각각 `width` 와 `height` 멤버 변수에 직접 대입하여 초기화한다. 초기화 완료 후 `cout` 을 통해 밑변과 높이를 출력하여 객체가 생성되었음을 알린다. 위임 생성자가 호출될 때도 결국 이 타겟 생성자가 실행되므로 출력문은 항상 이 생성자에서만 발생한다.

---

## 4. 소멸자 구현 - ~Triangle()

객체가 메모리에서 소멸되기 직전에 자동으로 호출된다. `cout` 을 통해 해당 객체의 밑변과 높이를 출력하여 소멸되었음을 알린다. 소멸 순서는 생성된 순서의 역순(LIFO)으로, 나중에 생성된 객체가 먼저 소멸된다.

---

## 5. getArea() 함수 구현

삼각형의 넓이 공식인 `(밑변 × 높이) ÷ 2` 를 계산하여 `double` 형으로 반환한다. 정수 나눗셈으로 인한 소수점 손실을 방지하기 위해 `1.0 / 2.0` 으로 표현하여 실수 연산이 되도록 처리한다.

---

## 6. main 함수

### tri1 객체 생성

`Triangle tri1;` 은 매개변수 없는 위임 생성자를 호출한다. 위임 생성자는 내부적으로 타겟 생성자 `Triangle(1, 1)` 을 호출하여 `width = 1`, `height = 1` 로 초기화한다. 이후 `tri1.getArea()` 를 호출하여 면적을 계산하고 결과를 출력한다. 면적은 `(1 × 1) ÷ 2 = 0.5` 이다.

### tri2 객체 생성

`Triangle tri2(2, 4);` 는 타겟 생성자를 직접 호출하여 `width = 2`, `height = 4` 로 초기화한다. 이후 `tri2.getArea()` 를 호출하여 면적 `(2 × 4) ÷ 2 = 4.0` 을 출력한다.

### 프로그램 종료

`return 0;` 이 실행되어 main 함수가 종료되면 지역 객체들이 생성된 역순으로 소멸된다. `tri2` 가 먼저 소멸된 후 `tri1` 이 소멸되며, 각 소멸 시점에 소멸자가 자동 호출되어 소멸 메시지가 출력된다.

---

## 7. 실행 결과

```
밑변 1 높이 1 인 삼각형 생성
삼각형의 면적은 0.5
밑변 2 높이 4 인 삼각형 생성
삼각형의 면적은 4
밑변 2 높이 4 인 삼각형 소멸
밑변 1 높이 1 인 삼각형 소멸
```

## 8. 핵심 개념 정리

| 개념 | 설명 |
|:----:|:-----|
| **위임 생성자** | 같은 클래스의 다른 생성자를 멤버 초기화 리스트에서 호출하여 초기화를 위임하는 생성자 |
| **타겟 생성자** | 위임 생성자로부터 실제 초기화 작업을 위임받아 수행하는 생성자 |
| **멤버 초기화 리스트** | 생성자 정의부에서 `: 멤버변수(값)` 형태로 멤버 변수를 초기화하는 문법 |
| **소멸자** | 객체 소멸 직전에 자동 호출되며, 클래스당 하나만 존재하고 매개변수와 리턴 타입이 없음 |
| **LIFO 소멸 순서** | 지역 객체는 스택 구조에 따라 나중에 생성된 객체가 먼저 소멸됨 |
