# ch5-2 실습과제 3 — swap 함수 (참조에 의한 호출)

---

## 문제

두 정수를 입력받아 교환하여 출력하는 프로그램을 작성하시오.  
두 정수를 교환하는 부분은 반드시 **참조에 의한 호출**을 이용한 `swap` 함수로 작성하시오.  
함수의 **선언, 호출, 정의**를 모두 사용할 것.

---

## 코드 설명

### 함수 선언 (프로토타입)

```cpp
void swap(int& a, int& b);
```

`main` 함수보다 앞에 프로토타입을 선언하여 컴파일러가 함수를 인식하도록 한다.  
`int& a`, `int& b`는 참조 매개변수임을 명시한다.

---

### 함수 호출

```cpp
swap(x, y);
```

값에 의한 호출과 동일한 문법이지만, 내부적으로 `int& a = x; int& b = y;`가 수행된다.  
`a`는 `x`의 별명, `b`는 `y`의 별명이 된다.

---

### 함수 정의 — 교환 동작

```cpp
void swap(int& a, int& b)
{
    int tmp;
    tmp = a;  // tmp에 x 값(15) 저장
    a = b;    // x에 y 값(22) 대입 → x = 22
    b = tmp;  // y에 tmp 값(15) 대입 → y = 15
}
```

`a`, `b`가 각각 `x`, `y`의 별명이므로, 교환 결과가 `main` 함수의 원본 변수에 그대로 반영된다.

### 메모리 흐름

```
(1) swap 호출 전       (2) swap 호출 직후     (3) swap 실행 후
  x = 15                x = 15                x = 22
  y = 22                y = 22                y = 15
                        a (x의 별명)
                        b (y의 별명)
```

참조 매개변수 `a`, `b`는 이름만 생성되고 별도의 메모리 공간은 할당되지 않는다.
