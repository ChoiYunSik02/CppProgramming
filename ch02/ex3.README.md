# 1장 실습과제 3 — 특수문자 출력

---

## 코드 설명

| 라인 | 설명 |
|------|------|
| `#include <iostream>` | 표준 입출력 스트림 헤더 포함 |
| `using namespace std;` | `std::cout` 대신 `cout` 으로 줄여 쓸 수 있게 선언 |
| `\"` | 이스케이프 시퀀스 — 큰따옴표(`"`) 출력 |
| `\\` | 이스케이프 시퀀스 — 역슬래시(`\`, 한글 키보드에서 ₩) 출력 |
| `endl` | 줄바꿈 출력 |

### 이스케이프 시퀀스 정리

| 이스케이프 시퀀스 | 출력 문자 |
|------------------|-----------|
| `\"` | `"` (큰따옴표) |
| `\\` | `\` (역슬래시 / ₩) |
| `\n` | 줄바꿈 |
| `\t` | 탭 |
