# 2-1장 실습과제4 — 다이아몬드 별 패턴 출력

---

## 코드 설명

| 라인 | 설명 |
|------|------|
| `int size = 7;` | X자 패턴의 크기를 7x7로 설정 |
| `for (int i = 0; i < size; i++)` | 행(row)을 0부터 6까지 순서대로 반복 |
| `for (int j = 0; j < size; j++)` | 각 행에서 열(column)을 0부터 6까지 반복 |
| `i == j` | **주 대각선** 조건 — 행과 열 번호가 같은 위치 (좌상→우하 방향) |
| `i + j == size - 1` | **부 대각선** 조건 — 행과 열의 합이 6인 위치 (우상→좌하 방향) |
| `cout << "*"` | 두 조건 중 하나라도 해당하면 별(*) 출력 |
| `cout << " "` | 두 조건 모두 해당하지 않으면 공백 출력 |
| `cout << endl` | 한 행이 끝나면 줄바꿈 |
