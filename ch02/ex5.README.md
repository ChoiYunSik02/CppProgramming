# 2-1장 87페이지 20번 — 별 삼각형 출력

---

## 코드 설명

| 라인 | 설명 |
|------|------|
| `for(int n=0; n<4; n++)` | 행을 0부터 3까지 총 4번 반복 (바깥 루프) |
| `for(int i=0; i<=n; i++)` | 현재 행 번호 n만큼 별을 반복 출력 (안쪽 루프) |
| `cout << "*"` | 별 1개 출력 |
| `cout << endl` | 한 행이 끝나면 줄바꿈 |

### 반복 흐름

| n값 | 안쪽 루프 반복 횟수 | 출력 |
|-----|-------------------|------|
| 0 | 1회 | `*` |
| 1 | 2회 | `**` |
| 2 | 3회 | `***` |
| 3 | 4회 | `****` |
