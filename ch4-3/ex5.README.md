# 실습과제 5 - Sphere 클래스 설명

## 클래스 구성
- `radius`: 반지름 저장
- `setRadius()`: 반지름 멤버 변수 설정
- `getVolume()`: 구의 부피 계산

> **부피 공식:** V = (4/3) * π * r³

---

## 핵심 개념

- **동적 배열 생성 (변수 크기 할당)**
```cpp
Sphere* arr = new Sphere[n];
```
- 사용자 입력 기반 크기 결정 및 값 처리
- 인덱스를 이용한 배열 순회 출력

## 결론
`new` 연산자를 이용해 동적 배열을 생성하면 실행 중 입력되는 개수(`n`)에 따라 유연하게 객체 배열을 생성할 수 있으며, 마지막에 메모리 해제(`delete[]`)를 해주어 안정적으로 메모리를 관리해야 한다.
