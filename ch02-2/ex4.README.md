# ch2-2 실습과제4번 — 실수 5개 입력 후 양수 합산
교재 88페이지 4번 문제를 푸시오.
---

## 코드 설명

| 라인 | 설명 |
|------|------|
| `double num, sum = 0;` | 입력값 저장용 `num`, 합계 저장용 `sum` 선언 (실수 합산이므로 double 사용) |
| `for (int i = 0; i < 5; i++)` | 5번 반복하여 실수를 하나씩 입력받음 |
| `cin >> num;` | 실수 1개를 입력받아 num에 저장 |
| `if (num > 0)` | 입력값이 양수인 경우에만 합산 (0과 음수는 제외) |
| `sum += num;` | 양수일 때 sum에 누적 |
| `cout << "양수 합은 " << sum` | 최종 양수 합계 출력 |
