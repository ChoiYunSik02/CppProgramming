# ch2-2 실습과제5 — 문자열 각 문자 분리 출력
교재 89페이지 8-(1)번문제를 푸시오
---
## 코드 설명

| 라인 | 설명 |
|------|------|
| `#include <cstring>` | `strlen()` 함수 사용을 위한 헤더 포함 |
| `char text[100];` | 최대 100글자의 C-스타일 문자 배열 선언 |
| `cin >> text;` | 빈칸 없는 문자열을 입력받아 text에 저장 |
| `strlen(text)` | 문자열의 길이(글자 수)를 반환 |
| `text[i]` | i번째 문자에 접근 |
| `cout << text[i] << " "` | 각 문자 뒤에 공백을 붙여 출력 |
