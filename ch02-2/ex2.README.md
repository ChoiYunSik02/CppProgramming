# 2장 실습과제 2 — 함수 Y = x² + 2x - 5 계산

---

## 코드 설명

| 라인 | 설명 |
|------|------|
| `int x, y;` | 정수형 변수 x(입력값), y(결과값) 선언 |
| `cout << "정수 x를 입력 하시오: ";` | 입력 안내 문구 출력 |
| `cin >> x;` | 키보드에서 정수를 입력받아 x에 저장 |
| `y = x * x + 2 * x - 5;` | 수식 Y = x² + 2x - 5 계산 |
| `cout << "x=" << x << ...` | x값과 계산된 y값을 출력 |

### 수식 검증 (x = 5 일 때)

```
Y = 5² + 2×5 - 5
  = 25 + 10 - 5
  = 30
```
