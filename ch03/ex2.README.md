# 3-1장 실습과제 2번


## 소스코드 설명 

### 1. 헤더 및 네임스페이스

| 줄 | 코드 | 설명 |
|---|---|---|
| 1 | `#include <iostream>` | 표준 입출력 라이브러리를 포함한다. `cout`, `endl` 등을 사용하기 위해 필요하다. |
| 2 | `using namespace std;` | `std::` 접두사 없이 표준 라이브러리의 이름을 바로 사용할 수 있게 한다. 예) `std::cout` → `cout` |

---

### 2. 클래스 선언부

| 줄 | 코드 | 설명 |
|---|---|---|
| 3 | `class Triangle {` | `Triangle`이라는 이름의 클래스를 선언하기 시작한다. 클래스는 데이터(멤버 변수)와 동작(멤버 함수)을 하나로 묶는 사용자 정의 타입이다. |
| 4 | `public:` | 이 아래에 선언된 멤버들은 **클래스 외부에서도 접근 가능**함을 나타내는 접근 지정자이다. |
| 5 | `int width;` | 삼각형의 **밑변(너비)**을 저장하는 정수형 멤버 변수를 선언한다. |
| 6 | `int height;` | 삼각형의 **높이**를 저장하는 정수형 멤버 변수를 선언한다. |
| 7 | `int getArea();` | 삼각형의 넓이를 반환하는 멤버 함수를 **선언(프로토타입)**만 한다. 실제 구현 코드는 클래스 블록 밖에서 작성한다. |
| 8 | `};` | 클래스 선언 블록을 닫는다. C++에서는 클래스의 닫는 중괄호 뒤에 반드시 **세미콜론(`;`)** 을 붙여야 한다. |

---

### 3. 멤버 함수 구현부

| 줄 | 코드 | 설명 |
|---|---|---|
| 9 | `int Triangle::getArea() {` | `Triangle` 클래스에 속한 `getArea()` 함수를 **클래스 외부에서 정의**한다. `::` 는 **범위 지정 연산자(scope resolution operator)** 로, 이 함수가 `Triangle` 클래스의 멤버임을 명시한다. |
| 10 | `return 0.5 * width * height;` | 삼각형 넓이 공식 **(밑변 × 높이 × 0.5)** 을 계산하여 반환한다. `0.5`가 `double`형이므로 계산 결과도 `double`이 되지만, 반환 타입이 `int`이므로 소수점 이하는 **자동으로 버려진다(암묵적 형변환)**. |
| 11 | `}` | `getArea()` 함수의 구현 블록을 닫는다. |

---

### 4. `main()` 함수

| 줄 | 코드 | 설명 |
|---|---|---|
| 12 | `int main() {` | 프로그램의 **진입점(entry point)** 인 `main` 함수를 정의한다. 프로그램 실행 시 이 함수부터 시작된다. |
| 13 | `Triangle tra;` | `Triangle` 타입의 **객체(object)** `tra`를 생성한다. 이 시점에서 `width`, `height`는 초기화되지 않은 쓰레기 값을 가진다. |
| 14 | `tra.width = 7;` | 객체 `tra`의 멤버 변수 `width`에 **7**을 대입한다. `.` 연산자(멤버 접근 연산자)를 사용해 객체의 멤버에 접근한다. |
| 15 | `tra.height = 8;` | 객체 `tra`의 멤버 변수 `height`에 **8**을 대입한다. |
| 16 | `cout << "삼각형의 면적은 " << tra.getArea() << endl;` | `tra.getArea()`를 호출해 넓이를 계산하고 (`0.5 × 7 × 8 = 28`), 결과를 화면에 출력한다. `endl`은 줄 바꿈과 출력 버퍼의 flush를 수행한다. |
| 17 | `}` | `main()` 함수의 블록을 닫으며 프로그램이 종료된다. |

## 핵심 개념 정리

| 개념 | 설명 |
|---|---|
| **클래스 (class)** | 멤버 변수와 멤버 함수를 하나로 묶는 사용자 정의 타입 |
| **객체 (object)** | 클래스를 통해 실제로 생성된 실체 (예: `tra`) |
| **접근 지정자 `public`** | 클래스 외부에서도 멤버에 자유롭게 접근 가능하도록 허용 |
| **멤버 접근 연산자 `.`** | 객체를 통해 해당 클래스의 멤버 변수나 멤버 함수에 접근할 때 사용 |
| **범위 지정 연산자 `::`** | 클래스 외부에서 멤버 함수를 정의할 때 어느 클래스 소속인지 명시 |
| **암묵적 형변환** | `0.5 * width * height`의 `double` 결과가 `int` 반환 타입에 맞게 자동으로 정수로 변환됨 |
