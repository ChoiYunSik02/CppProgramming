## 줄별 설명

### 1. 헤더 및 네임스페이스

| 줄 | 코드 | 설명 |
|---|---|---|
| 1 | `#include <iostream>` | 표준 입출력 라이브러리를 포함한다. `cout`, `endl` 등을 사용하기 위해 필요하다. |
| 2 | `using namespace std;` | `std::` 접두사 없이 표준 라이브러리의 이름을 바로 사용할 수 있게 한다. 예) `std::cout` → `cout` |

---

### 2. 클래스 선언부

| 줄 | 코드 | 설명 |
|---|---|---|
| 3 | `class Sphere {` | `Sphere`라는 이름의 클래스를 선언한다. 구(球)를 표현하는 데이터와 기능을 하나로 묶는 사용자 정의 타입이다. |
| 4 | `public:` | 이 아래에 선언된 멤버들은 **클래스 외부에서도 접근 가능**함을 나타내는 접근 지정자이다. |
| 5 | `int radius;` | 구의 **반지름**을 저장하는 정수형 멤버 변수를 선언한다. |
| 6 | `double Volume();` | 구의 **부피**를 계산하여 `double`형으로 반환하는 멤버 함수를 선언(프로토타입)한다. |
| 7 | `double SurfaceArea();` | 구의 **표면적**을 계산하여 `double`형으로 반환하는 멤버 함수를 선언(프로토타입)한다. |

---

### 3. 멤버 함수 구현부

| 줄 | 코드 | 설명 |
|---|---|---|
| 10 | `double Sphere::Volume() {` | `Sphere` 클래스에 속한 `Volume()` 함수를 **클래스 외부에서 정의**한다. `::` 는 범위 지정 연산자로, 이 함수가 `Sphere` 클래스의 멤버임을 명시한다. |
| 11 | `return (4.0 / 3.0) * 3.14 * radius * radius * radius;` | 구의 부피 공식 **(4/3 × π × r³)** 을 계산하여 반환한다. `4.0 / 3.0`으로 작성하여 정수 나눗셈을 방지하고, `radius`를 세 번 곱해 **r³** 을 표현한다. |
| 14 | `double Sphere::SurfaceArea() {` | `Sphere` 클래스에 속한 `SurfaceArea()` 함수를 **클래스 외부에서 정의**한다. |
| 15 | `return 4 * 3.14 * radius * radius;` | 구의 표면적 공식 **(4 × π × r²)** 을 계산하여 반환한다. `radius`를 두 번 곱해 **r²** 을 표현한다. |

---

### 4. `main()` 함수

| 줄 | 코드 | 설명 |
|---|---|---|
| 18 | `int main() {` | 프로그램의 **진입점(entry point)** 인 `main` 함수를 정의한다. |
| 19 | `Sphere gu;` | `Sphere` 타입의 **객체(object)** `gu`를 생성한다. |
| 20 | `gu.radius = 5;` | 객체 `gu`의 멤버 변수 `radius`에 **5**를 대입한다. 이후 부피/표면적 계산에 이 값이 사용된다. |
| 21 | `cout << "구의 부피는 " << gu.Volume() << endl;` | `gu.Volume()`을 호출해 부피를 계산하고 결과를 화면에 출력한다. |
| 22 | `cout << "구의 표면적은 " << gu.SurfaceArea() << endl;` | `gu.SurfaceArea()`를 호출해 표면적을 계산하고 결과를 화면에 출력한다. |
| 23 | `return 0;` | `main` 함수가 정상적으로 종료되었음을 운영체제에 알린다. |

## 핵심 개념 정리

| 개념 | 설명 |
|---|---|
| **클래스 (class)** | 멤버 변수와 멤버 함수를 하나로 묶는 사용자 정의 타입 |
| **객체 (object)** | 클래스를 통해 실제로 생성된 실체 (예: `gu`) |
| **접근 지정자 `public`** | 클래스 외부에서도 멤버에 자유롭게 접근 가능하도록 허용 |
| **범위 지정 연산자 `::`** | 클래스 외부에서 멤버 함수를 정의할 때 어느 클래스 소속인지 명시 |
| **`4.0 / 3.0`** | 정수끼리 나누면 소수점이 버려지므로(`4/3 = 1`), `double`형 나눗셈을 위해 실수 리터럴로 작성 |
