## 소스코드 설명

### 1. 헤더 및 네임스페이스

| 줄 | 코드 | 설명 |
|---|---|---|
| 1 | `#include <iostream>` | 표준 입출력 라이브러리를 포함한다. `cout`, `endl` 등을 사용하기 위해 필요하다. |
| 2 | `using namespace std;` | `std::` 접두사 없이 표준 라이브러리의 이름을 바로 사용할 수 있게 한다. |

---

### 2. 클래스 선언부

| 줄 | 코드 | 설명 |
|---|---|---|
| 3 | `class Rectangle {` | `Rectangle`이라는 이름의 클래스를 선언한다. 사각형을 표현하는 데이터와 기능을 하나로 묶는 사용자 정의 타입이다. |
| 4 | `public:` | 이 아래에 선언된 멤버들은 **클래스 외부에서도 접근 가능**함을 나타내는 접근 지정자이다. |
| 5 | `int x;` | 사각형 **좌측 상단의 x좌표**를 저장하는 정수형 멤버 변수를 선언한다. |
| 6 | `int y;` | 사각형 **좌측 상단의 y좌표**를 저장하는 정수형 멤버 변수를 선언한다. |
| 7 | `int width;` | 사각형의 **가로 길이(밑변)**를 저장하는 정수형 멤버 변수를 선언한다. |
| 8 | `int height;` | 사각형의 **세로 길이(높이)**를 저장하는 정수형 멤버 변수를 선언한다. |
| 9 | `int getArea();` | 사각형의 **면적**을 계산하여 반환하는 멤버 함수를 선언(프로토타입)한다. |
| 10 | `int Perimeter();` | 사각형의 **둘레 길이**를 계산하여 반환하는 멤버 함수를 선언(프로토타입)한다. |
| 11 | `int CoordinateX();` | 사각형 **우측 하단의 x좌표**를 계산하여 반환하는 멤버 함수를 선언(프로토타입)한다. |
| 12 | `int CoordinateY();` | 사각형 **우측 하단의 y좌표**를 계산하여 반환하는 멤버 함수를 선언(프로토타입)한다. |

---

### 3. 멤버 함수 구현부

| 줄 | 코드 | 설명 |
|---|---|---|
| 15 | `int Rectangle::getArea() {` | `Rectangle` 클래스에 속한 `getArea()` 함수를 클래스 외부에서 정의한다. |
| 16 | `return width * height;` | 사각형의 면적 공식 **(가로 × 세로)** 를 계산하여 반환한다. |
| 18 | `int Rectangle::Perimeter() {` | `Rectangle` 클래스에 속한 `Perimeter()` 함수를 클래스 외부에서 정의한다. |
| 19 | `return (width + height) * 2;` | 사각형의 둘레 공식 **((가로 + 세로) × 2)** 를 계산하여 반환한다. |
| 21 | `int Rectangle::CoordinateX() {` | `Rectangle` 클래스에 속한 `CoordinateX()` 함수를 클래스 외부에서 정의한다. |
| 22 | `return x + width;` | 우측 하단의 **x좌표**를 계산한다. 좌측 상단 x좌표에서 가로 길이만큼 더하면 우측 하단 x좌표가 된다. |
| 24 | `int Rectangle::CoordinateY() {` | `Rectangle` 클래스에 속한 `CoordinateY()` 함수를 클래스 외부에서 정의한다. |
| 25 | `return y - height;` | 우측 하단의 **y좌표**를 계산한다. y축이 위로 갈수록 증가하는 좌표계 기준으로, 아래로 내려갈수록 y값이 감소하므로 좌측 상단 y좌표에서 높이를 **뺀다**. |

### 4. `main()` 함수

| 줄 | 코드 | 설명 |
|---|---|---|
| 28 | `int main() {` | 프로그램의 **진입점(entry point)** 인 `main` 함수를 정의한다. |
| 29 | `Rectangle rect;` | `Rectangle` 타입의 **객체(object)** `rect`를 생성한다. |
| 30 | `rect.x = 1;` | 객체 `rect`의 멤버 변수 `x`에 **1**을 대입한다. 사각형 좌측 상단의 x좌표이다. |
| 31 | `rect.y = 2;` | 객체 `rect`의 멤버 변수 `y`에 **2**를 대입한다. 사각형 좌측 상단의 y좌표이다. |
| 32 | `rect.width = 3;` | 객체 `rect`의 멤버 변수 `width`에 **3**을 대입한다. 사각형의 가로 길이이다. |
| 33 | `rect.height = 4;` | 객체 `rect`의 멤버 변수 `height`에 **4**를 대입한다. 사각형의 세로 길이이다. |
| 34 | `cout << "사각형의 면적은 " << rect.getArea() << endl;` | `rect.getArea()`를 호출해 면적을 계산하고 (`3 × 4 = 12`), 결과를 화면에 출력한다. |
| 35 | `cout << "사각형의 둘레길이는 " << rect.Perimeter() << endl;` | `rect.Perimeter()`를 호출해 둘레를 계산하고 (`(3 + 4) × 2 = 14`), 결과를 화면에 출력한다. |
| 36 | `cout << "사각형의 우측하단의 좌표는 " << "(" << rect.CoordinateX() << ", " << rect.CoordinateY() << ")" << endl;` | `rect.CoordinateX()`와 `rect.CoordinateY()`를 각각 호출해 우측 하단 좌표를 계산하고 (`1+3=4`, `2-4=-2`), `(4, -2)` 형식으로 출력한다. |

## 핵심 개념 정리

| 개념 | 설명 |
|---|---|
| **클래스 (class)** | 멤버 변수와 멤버 함수를 하나로 묶는 사용자 정의 타입 |
| **객체 (object)** | 클래스를 통해 실제로 생성된 실체 (예: `rect`) |
| **접근 지정자 `public`** | 클래스 외부에서도 멤버에 자유롭게 접근 가능하도록 허용 |
| **범위 지정 연산자 `::`** | 클래스 외부에서 멤버 함수를 정의할 때 어느 클래스 소속인지 명시 |
| **y좌표 계산 `y - height`** | y축이 위로 갈수록 증가하는 좌표계에서 아래 방향은 y값이 감소하므로 높이를 빼서 우측 하단 y좌표를 구함 |
