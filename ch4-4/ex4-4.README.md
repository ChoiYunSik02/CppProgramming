# 4장 실습문제 8번 - 소스코드 설명

---

```cpp
#include <iostream>
```
`cin`, `cout` 사용을 위한 헤더 파일

```cpp
#include <string>
```
`string` 클래스 사용을 위한 헤더 파일

```cpp
using namespace std;
```
`std::` 접두사 없이 사용 가능하게 설정

---

```cpp
while (true)
```
`"그만"` 입력 전까지 계속 URL을 입력받아 파싱

```cpp
string url;
getline(cin, url, '\n');
```
공백 포함 한 줄 전체를 `url`에 저장

```cpp
if (url == "그만") break;
```
`"그만"` 입력 시 반복문 탈출 → 프로그램 종료

---

```cpp
int protoEnd = url.find("://");
```
`"://"`가 시작되는 인덱스를 찾아 저장. 예) `"https://..."` → `protoEnd = 5`

```cpp
string protocol = url.substr(0, protoEnd);
```
인덱스 0부터 `protoEnd` 직전까지 추출 → 프로토콜 이름. 예) `substr(0, 5)` → `"https"`

```cpp
int hostStart = protoEnd + 3;
```
`"://"`는 3글자이므로 `+3`을 하면 호스트가 시작하는 인덱스

```cpp
int colonPos = url.find(':', hostStart);
```
`hostStart`부터 탐색해 포트 구분자 `':'`의 위치를 찾음. `hostStart`를 시작점으로 지정해야 앞의 `"://"`속 `':'`를 다시 찾지 않음

```cpp
int slashPos = url.find('/', hostStart);
```
`hostStart`부터 탐색해 경로 시작 `'/'`의 위치를 찾음

---

```cpp
if (colonPos != (int)string::npos && colonPos < slashPos)
```
`colonPos`가 유효하고(찾았고) `slashPos`보다 앞에 있으면 포트 번호가 존재하는 URL. `string::npos`는 `find()`가 찾지 못했을 때 반환하는 특수값이며 `(int)` 캐스팅으로 안전하게 비교

```cpp
host = url.substr(hostStart, colonPos - hostStart);
```
`hostStart`부터 `':'` 직전까지 추출 → 호스트 이름

```cpp
int portEnd = (slashPos != (int)string::npos) ? slashPos : (int)url.length();
```
`'/'`가 있으면 `slashPos`를, 없으면 문자열 끝을 포트의 끝 위치로 설정

```cpp
port = url.substr(colonPos + 1, portEnd - colonPos - 1);
```
`':'` 다음 글자부터 경로 직전까지 추출 → 포트 번호 문자열

```cpp
rest = (slashPos != (int)string::npos) ? url.substr(slashPos) : "/";
```
`'/'`가 있으면 `slashPos`부터 끝까지 추출, 없으면 기본값 `"/"` 사용

---

```cpp
} else {
```
포트 번호가 없는 URL 처리 분기

```cpp
host = url.substr(hostStart, slashPos - hostStart);
```
`hostStart`부터 `'/'` 직전까지 추출 → 호스트 이름

```cpp
rest = url.substr(slashPos);
```
`'/'`부터 끝까지 추출 → 나머지 경로

```cpp
port = "80";
```
포트가 명시되지 않았으므로 HTTP 기본 포트 `80`으로 설정
