# 4장 실습문제 4번 - 소스코드 설명

---

```cpp
#include <iostream>
```
`cin`, `cout` 사용을 위한 헤더 파일

```cpp
#include <string>
```
`string` 클래스 사용을 위한 헤더 파일

```cpp
using namespace std;
```
`std::` 접두사 없이 사용 가능하게 설정

---

```cpp
string s;
```
사용자가 입력한 원본 문자열을 저장하는 변수

```cpp
getline(cin, s, '\n');
```
공백 포함 한 줄 전체를 `s`에 저장

```cpp
string dest = "";
```
필터링된 결과를 담을 빈 문자열

---

```cpp
for (int i = 0; i < (int)s.length(); i++)
```
인덱스 0부터 문자열 끝까지 한 글자씩 순회

```cpp
char ch = s[i];
```
`i`번째 문자를 `ch`에 저장

```cpp
if (isalpha(ch) || ch == ' ')
```
`isalpha(ch)` : `ch`가 영어 대문자 또는 소문자이면 `true`. `ch == ' '` : `ch`가 빈칸이면 `true`. 둘 중 하나라도 해당되면 결과에 포함

```cpp
dest += ch;
```
조건을 만족하는 문자만 결과 문자열에 이어붙임. `'`, `!`, `2`, `5` 등 숫자/특수문자는 조건 불충족으로 제외됨

```cpp
cout << dest << endl;
```
필터링된 결과 문자열 출력
