# 4장 실습문제 2번 - 소스코드 설명

---

```cpp
#include <iostream>
```
`cin`, `cout` 사용을 위한 헤더 파일

```cpp
using namespace std;
```
`std::` 접두사 없이 사용 가능하게 설정

---

```cpp
int n;
```
물품 개수를 저장하는 변수. 입력받은 값에 따라 배열 크기가 결정됨

```cpp
cin >> n;
```
물품 개수 입력받기

```cpp
int* prices = new int[n];
```
`n`개짜리 정수 배열을 힙(heap)에 동적 할당. `n`이 실행 중에 결정되므로 `int arr[n]` 대신 `new` 사용

---

```cpp
for (int i = 0; i < n; i++)
```
인덱스 0부터 n-1까지 반복하며 가격 입력

```cpp
cin >> prices[i];
```
입력값을 배열의 `i`번째 원소에 직접 저장

---

```cpp
int minPrice = prices[0];
int maxPrice = prices[0];
```
첫 번째 원소를 최솟값/최댓값의 초기값으로 설정

```cpp
for (int i = 1; i < n; i++)
```
0번은 이미 초기값으로 사용했으므로 1번 인덱스부터 시작

```cpp
if (prices[i] < minPrice) minPrice = prices[i];
```
현재 원소가 지금까지의 최솟값보다 작으면 갱신

```cpp
if (prices[i] > maxPrice) maxPrice = prices[i];
```
현재 원소가 지금까지의 최댓값보다 크면 갱신

---

```cpp
delete[] prices;
```
`new[]`로 할당한 배열 메모리 반환. `delete`(대괄호 없이)를 쓰면 첫 번째 원소만 해제되어 메모리 누수 발생
