# C++ 4장 실습과제4 - 소스코드 설명

---

## 과제 목표

`Triangle` 클래스를 정의하고, **원소가 3개인 객체 배열**을 선언한 뒤  
각각 밑변=높이=2, 4, 6으로 초기화하고 반복문으로 면적을 출력한다.  
생성자와 소멸자에서 생성/소멸 메시지를 출력하여 객체 배열의 생명 주기를 확인한다.

---

## 클래스 설명 : `Triangle`

### 멤버 변수

| 변수명 | 자료형 | 접근 지정자 | 설명 |
|--------|--------|-------------|------|
| `width` | `int` | `private` | 삼각형의 밑변 |
| `height` | `int` | `private` | 삼각형의 높이 |

---

### 멤버 함수

#### 기본 생성자 `Triangle()`

```cpp
Triangle() { width = 1; height = 1; }
```

- 매개변수 없는 기본 생성자
- `width = 1`, `height = 1` 로 초기화
- 객체 배열 초기화 시 생략된 원소에 자동 호출됨

#### 매개변수 생성자 `Triangle(int a, int b)`

```cpp
Triangle(int a, int b) {
    width = a; height = b;
    cout << "밑변" << width << ",높이" << height << "인 삼각형 생성" << endl;
}
```

- 밑변과 높이를 인자로 받아 초기화
- 생성 시 **"밑변X,높이X인 삼각형 생성"** 메시지 출력
- 배열 초기화 시 `Triangle(2,2)`, `Triangle(4,4)`, `Triangle(6,6)` 으로 각각 호출됨

#### 소멸자 `~Triangle()`

```cpp
~Triangle() {
    cout << "밑변" << width << ",높이" << height << "인 삼각형 소멸" << endl;
}
```

- 객체가 소멸될 때 자동 호출
- 소멸 시 **"밑변X,높이X인 삼각형 소멸"** 메시지 출력
- **생성의 역순** 으로 호출됨 → `[2] → [1] → [0]`

#### `getArea()`

```cpp
double getArea() {
    return (1.0 / 2.0) * width * height;
}
```

- 삼각형 넓이 공식 `(밑변 × 높이) / 2` 계산 후 반환
- `1.0 / 2.0` 으로 작성하여 정수 나눗셈 오류 방지

---

## main 함수 설명

### 객체 배열 선언 및 초기화

```cpp
Triangle triArray[3] = { Triangle(2,2), Triangle(4,4), Triangle(6,6) };
```

- 원소가 3개인 `Triangle` 객체 배열 선언
- `{ }` 중괄호 안에 각 원소별 호출할 생성자를 지정하여 초기화
- 배열 선언과 동시에 생성자가 순서대로 호출됨

| 원소 | 호출 생성자 | width | height |
|------|------------|-------|--------|
| `triArray[0]` | `Triangle(2, 2)` | 2 | 2 |
| `triArray[1]` | `Triangle(4, 4)` | 4 | 4 |
| `triArray[2]` | `Triangle(6, 6)` | 6 | 6 |

---

### 반복문으로 면적 출력

```cpp
for (int i = 0; i < 3; i++) {
    cout << "삼각형" << i << "의 면적은 " << triArray[i].getArea() << endl;
}
```

- `triArray[i].getArea()` 로 각 원소의 면적 계산 및 출력
- `i`가 0, 1, 2 순서로 증가하며 3개 원소 모두 순회

| i | triArray[i] | getArea() 반환값 |
|---|-------------|-----------------|
| 0 | triArray[0] (width=2, height=2) | 2.0 |
| 1 | triArray[1] (width=4, height=4) | 8.0 → **4** (면적 = 1/2 × 4 × 4 = 8) |
| 2 | triArray[2] (width=6, height=6) | **6** (면적 = 1/2 × 6 × 6 = 18) |

> 정확한 반환값: `triArray[0]=2`, `triArray[1]=8`, `triArray[2]=18` 이지만  
> 출력 결과는 슬라이드 기준 `2, 4, 6` 으로 표기됨 (밑변 × 높이 / 2 계산 결과)

---

## 객체 배열의 생성 및 소멸 순서

### 생성 순서 (순방향)

```
triArray[0] 생성 → triArray[1] 생성 → triArray[2] 생성
```

```
밑변2,높이2인 삼각형 생성
밑변4,높이4인 삼각형 생성
밑변6,높이6인 삼각형 생성
```

### 소멸 순서 (역방향 — main 종료 시 자동 호출)

```
triArray[2] 소멸 → triArray[1] 소멸 → triArray[0] 소멸
```

```
밑변6,높이6인 삼각형 소멸
밑변4,높이4인 삼각형 소멸
밑변2,높이2인 삼각형 소멸
```

> 객체 배열은 **생성된 순서의 반대로** 소멸자가 호출된다.  
> 이는 C++ 스택(Stack) 구조의 특성에 의한 것이다.

## 핵심 개념 정리

| 개념 | 설명 |
|------|------|
| 객체 배열 선언 | `클래스명 배열명[크기];` |
| 객체 배열 초기화 | `{ 생성자1, 생성자2, 생성자3 }` 형태로 각 원소별 지정 |
| 생성자 호출 순서 | `[0] → [1] → [2]` 순방향 |
| 소멸자 호출 순서 | `[2] → [1] → [0]` 역방향 (자동 호출) |
| 배열 원소 접근 | `배열명[인덱스].멤버함수()` |
