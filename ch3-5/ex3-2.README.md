```
Coffee 클래스
├── private 멤버변수

└── public 멤버함수

    └── show()                     → 재료 시각화 출력
```

## 3. 클래스 선언

```cpp
class Coffee {
private:
    int coffee;  // 커피
    int sugar;   // 설탕
    int milk;    // 우유
    int water;   // 물
public:
    Coffee();
    Coffee(int c, int s, int m, int w);
    void show();
};
```

- 멤버변수 4개 모두 **private** → 외부 직접 접근 불가 (캡슐화)
- 멤버함수는 **public** → 외부에서 호출 가능
- 클래스 내부에는 **선언만** 작성, 구현은 클래스 외부에서 분리 (인라인 함수 미사용)

---

## 4. 생성자 구현

### 4-1. 기본 생성자

```cpp
Coffee::Coffee() : coffee(10), sugar(0), milk(0), water(0) { }
```

- 인자 없이 객체 생성 시 호출
- **멤버 초기화 리스트**로 `coffee=10`, 나머지는 모두 `0`으로 초기화
- `espresso` 객체의 출력 결과(`coffee **********`)에서 기본값 유추

### 4-2. 매개변수 생성자

```cpp
Coffee::Coffee(int c, int s, int m, int w) : coffee(c), sugar(s), milk(m), water(w) { }
```

- 인자 4개(커피, 설탕, 우유, 물)를 받아 각 멤버변수 초기화
- 멤버 초기화 리스트만으로 처리 가능하므로 본문 `{}` 비어 있음

## 5. show() 함수 구현

```cpp
void Coffee::show() {
    cout << "coffee ";
    for (int i = 0; i < coffee; i++) cout << "*";
    cout << endl;

    cout << "sugar ";
    for (int i = 0; i < sugar; i++) cout << "*";
    cout << endl;

    cout << "milk ";
    for (int i = 0; i < milk; i++) cout << "*";
    cout << endl;

    cout << "water ";
    for (int i = 0; i < water; i++) cout << "*";
    cout << endl;
}
```

- 각 재료명을 출력한 뒤, **해당 값만큼 `*`를 반복 출력**
- 값이 `0`이면 for문이 0번 실행 → `*` 없이 재료명만 출력됨

### show() 동작 예시

| 멤버변수 값 | 출력 결과 |
|------------|-----------|
| `coffee=10` | `coffee **********` |
| `sugar=0` | `sugar ` |
| `coffee=3` | `coffee ***` |
| `sugar=7` | `sugar *******` |

## 6. main() 동작 흐름

```cpp
Coffee espresso;                   // 기본 생성자 호출
Coffee americano(5, 0, 0, 10);    // 매개변수 생성자 호출
Coffee cappucchino(5, 1, 5, 2);   // 매개변수 생성자 호출
Coffee mySweet(3, 7, 5, 5);       // 매개변수 생성자 호출
```

| 객체 | coffee | sugar | milk | water |
|------|--------|-------|------|-------|
| `espresso` | 10 | 0 | 0 | 0 |
| `americano` | 5 | 0 | 0 | 10 |
| `cappucchino` | 5 | 1 | 5 | 2 |
| `mySweet` | 3 | 7 | 5 | 5 |

> `americano`, `cappucchino`는 생성만 되고 `show()` 호출 없음 → 출력 결과에 나타나지 않음

```cpp
espresso.show();  // 에스프레소 재료 출력
cout << endl;     // 빈 줄 출력
mySweet.show();   // 나만의 커피 재료 출력
```

## 8. 핵심 개념 정리

| 개념 | 내용 |
|------|------|
| 캡슐화 | 멤버변수를 `private`으로 숨기고 `show()`로만 출력 |
| 생성자 오버로딩 | 인자 수에 따라 기본/매개변수 생성자 구분 호출 |
| 멤버 초기화 리스트 | `: coffee(10), sugar(0), ...` 형태로 멤버 초기화 |
| 클래스 외부 구현 | `Coffee::함수명()` 형태로 선언과 구현 분리 |
| for문 활용 | 멤버변수 값만큼 `*` 반복 출력으로 시각화 |
