```
Picture 클래스
├── private 멤버변수

    ├── getHeight()            → 세로 크기 반환
    └── getPlace()             → 장소 반환
```

## 3. 클래스 선언

```cpp
class Picture {
private:
    int width;       // 가로 크기
    int height;      // 세로 크기
    char place[20];  // 촬영 장소 (최대 19글자 + 널문자)
public:
    Picture();
    Picture(int a, int b, const char* p);
    int getWidth();
    int getHeight();
    const char* getPlace();
};
```

- 멤버변수는 **private** → 외부에서 직접 접근 불가 (캡슐화)
- 멤버함수는 **public** → 외부에서 호출 가능
- `place[20]` : `string` 대신 **char 배열** 사용 (C 스타일 문자열)
- 클래스 내부에는 **선언만** 작성, 구현은 클래스 외부에서 분리 (인라인 함수 미사용)

## 4. 생성자 구현

### 4-1. 기본 생성자

```cpp
Picture::Picture() : width(5), height(7) {
    strcpy(place, "모름");
}
```

- 아무 인자 없이 객체 생성 시 호출
- **멤버 초기화 리스트** (`: width(5), height(7)`)로 정수형 멤버 초기화
- `place`는 배열이므로 초기화 리스트 사용 불가 → 생성자 본문에서 `strcpy`로 초기화
- 초기 기본값 : 가로 **5**, 세로 **7**, 장소 **"모름"**

### 4-2. 매개변수 생성자

```cpp
Picture::Picture(int a, int b, const char* p) : width(a), height(b) {
    strcpy(place, p);
}
```

- 인자 3개(가로, 세로, 장소)를 받아 객체 초기화
- `const char* p` : 문자열 리터럴(`"한라산"`)을 안전하게 받기 위해 `const` 사용
- `strcpy(place, p)` : 전달받은 문자열을 `place` 배열에 복사

> ⚠️ `char` 배열은 `place = p` 처럼 `=` 대입이 불가능하므로 반드시 `strcpy` 사용

---

## 5. strcpy 설명

```
strcpy(목적지 배열, 복사할 문자열)
```

문자열을 **한 글자씩 복사**하고 마지막에 **널문자(`\0`)** 도 함께 복사

```
strcpy(place, "한라산")
→ place[0]='한', place[1]='라', place[2]='산', place[3]='\0'
```

## 6. getter 함수 구현

```cpp
int Picture::getWidth()  { return width; }
int Picture::getHeight() { return height; }
const char* Picture::getPlace() { return place; }
```

- private 멤버변수를 외부에서 읽을 수 있도록 값을 반환하는 함수
- `getPlace()`의 반환 타입은 `const char*` → 반환된 문자열을 외부에서 수정하지 못하도록 보호

## 7. main() 동작 흐름

```cpp
Picture pic;                   // 기본 생성자 호출
Picture mt(10, 14, "한라산");  // 매개변수 생성자 호출
```

| 객체 | 생성자 | width | height | place |
|------|--------|-------|--------|-------|
| `pic` | 기본 생성자 | 5 | 7 | "모름" |
| `mt` | 매개변수 생성자 | 10 | 14 | "한라산" |

```cpp
cout << pic.getWidth() << "x" << pic.getHeight() << " " << pic.getPlace() << endl;
cout << mt.getWidth()  << "x" << mt.getHeight()  << " " << mt.getPlace()  << endl;
```

### 실행 결과

```
5x7 모름
10x14 한라산
```
