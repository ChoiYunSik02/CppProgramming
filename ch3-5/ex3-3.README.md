## 1. 전체 구조 요약

```
Account 클래스
├── private 멤버변수
│   ├── user[20]  (소유자 이름)
│   └── balance   (잔액)
└── public 멤버함수
    ├── Account(const char* name) → 생성자
    ├── deposit(int money)        → 입금
    ├── withdraw(int money)       → 출금 (실제 출금액 반환)
    ├── getUser()                 → 이름 반환
    └── inquiry()                 → 잔액 반환
```

## 2. 헤더 및 전처리

```cpp
#define _CRT_SECURE_NO_WARNINGS
#include <iostream>
```

| 코드 | 설명 |
|------|------|

| `<iostream>` | `cout`, `endl` 사용을 위한 입출력 헤더 |
| `<cstring>` | `strcpy` 사용을 위한 C 문자열 헤더 |

## 3. 클래스 선언

```cpp
class Account {
private:
    char user[20];  // 소유자 이름 (최대 19글자 + 널문자)
    int balance;    // 잔액
public:
    Account(const char* name);
    void deposit(int money);
    int withdraw(int money);
    const char* getUser();
    int inquiry();
};
```

- 멤버변수는 **private** → 외부에서 직접 접근 불가 (캡슐화)
- 멤버함수는 **public** → 외부에서 호출 가능
- `user[20]` : `string` 대신 **char 배열** 사용 (C 스타일 문자열)
- 클래스 내부에는 **선언만** 작성, 구현은 클래스 외부에서 분리

---

## 4. 생성자 구현

```cpp
Account::Account(const char* name) : balance(0) {
    strcpy(user, name);
}
```

- **멤버 초기화 리스트**로 `balance=0` 초기화 (계좌 개설 시 잔액은 0)
- `strcpy(user, name)` : 전달받은 이름 문자열을 `user` 배열에 복사
- `char` 배열은 `=` 대입 불가 → 반드시 `strcpy` 사용

---

## 5. deposit() - 입금 함수

```cpp
void Account::deposit(int money) {
    balance += money;
}
```

- 매개변수로 받은 금액을 잔액에 **더하기**
- 반환값 없음 (`void`)

---

## 6. withdraw() - 출금 함수

```cpp
int Account::withdraw(int money) {
    if (money > balance) {   // 출금 요청액 > 잔액 일 때
        int temp = balance;  // 현재 잔액 임시 저장
        balance = 0;         // 잔액을 0으로 초기화
        return temp;         // 잔액만큼만 출금하여 리턴
    }
    balance -= money;        // 잔액이 충분하면 정상 출금
    return money;            // 요청한 출금액 리턴
}
```

이 문제의 **핵심 로직**으로, 두 가지 경우를 처리:

| 상황 | 처리 방식 | 반환값 |
|------|----------|--------|
| 요청액 > 잔액 | 잔액 전부 출금, `balance=0` | 기존 잔액 |
| 요청액 ≤ 잔액 | 정상 출금, `balance -= money` | 요청 금액 |

### 동작 예시

```
1차 출금: withdraw(15000)
  잔액 20000 ≥ 요청 15000 → 정상 출금
  balance = 20000 - 15000 = 5000
  return 15000

2차 출금: withdraw(8000)
  잔액 5000 < 요청 8000 → 잔액만큼 출금
  temp = 5000
  balance = 0
  return 5000   ← 8000이 아닌 5000 리턴!
```

## 7. getter 함수 구현

```cpp
const char* Account::getUser() {
    return user;
}

int Account::inquiry() {
    return balance;
}
```

| 함수 | 반환 타입 | 설명 |
|------|----------|------|
| `getUser()` | `const char*` | 이름 문자열 반환, `const`로 외부 수정 방지 |
| `inquiry()` | `int` | 현재 잔액 반환 |

## 8. main() 동작 흐름

```cpp
Account a("황수희");   // 생성자 호출: user="황수희", balance=0
a.deposit(20000);      // balance = 0 + 20000 = 20000
```

```cpp
// 1단계 출력
cout << a.getUser() << " 잔액은 " << a.inquiry() << endl;
// → "황수희 잔액은 20000"
```

```cpp
int money = a.withdraw(15000);  // 정상 출금 → money=15000, balance=5000
cout << money << "원 출금, ";
cout << a.getUser() << " 잔액은 " << a.inquiry() << endl;
// → "15000원 출금, 황수희 잔액은 5000"
```

```cpp
money = a.withdraw(8000);  // 잔액 5000 < 요청 8000 → money=5000, balance=0
cout << money << "원 출금, ";
cout << a.getUser() << " 잔액은 " << a.inquiry() << endl;
// → "5000원 출금, 황수희 잔액은 0"
```
