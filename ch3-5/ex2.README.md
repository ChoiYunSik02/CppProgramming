# 실습과제2 소스코드 설명

---

## 클래스 구조

```
Triangle
├── private
│   ├── int width   // 삼각형의 폭
│   └── int height  // 삼각형의 높이
└── public
    ├── Triangle()         // 생성자
    ├── ~Triangle()        // 소멸자
    ├── double getArea()   // 면적 반환
    ├── void setWidth(int) // 폭 설정
    └── void setHeight(int)// 높이 설정
```

---

## 멤버 변수 설명

| 변수명 | 자료형 | 접근 지정자 | 설명 |
|--------|--------|-------------|------|
| `width` | `int` | `private` | 삼각형의 폭 |
| `height` | `int` | `private` | 삼각형의 높이 |

- `private`으로 선언하여 외부에서 직접 접근 불가 → **정보은닉(캡슐화)** 구현

---

## 멤버 함수 설명

### 1. 생성자 `Triangle()`

```cpp
Triangle() {
    width = 1, height = 1;
    cout << "폭" << width << ", " << "높이" << height << " 인 삼각형 생성" << endl;
}
```

- 객체가 생성될 때 **자동으로 호출**되는 함수
- `width`와 `height`를 각각 **1로 초기화**
- 생성 메시지를 콘솔에 출력
- 클래스 선언부 내부에 구현되어 있으므로 **자동 인라인 함수**로 처리됨

### 2. 소멸자 `~Triangle()`

```cpp
~Triangle() {
    cout << "폭" << width << ", " << "높이" << height << " 인 삼각형 소멸" << endl;
}
```

- 객체가 소멸될 때 **자동으로 호출**되는 함수
- `main()` 함수가 종료되면서 지역 객체 `tri`가 메모리에서 해제될 때 실행됨
- 소멸 시점의 `width`, `height` 값(변경된 값인 3, 5)을 출력
- 마찬가지로 **자동 인라인 함수**로 처리됨

### 3. 면적 계산 함수 `getArea()`

```cpp
double getArea() { return (1.0 / 2.0) * width * height; }
```

- 삼각형의 넓이 공식 : **넓이 = (1/2) × 폭 × 높이** 적용
- `1.0 / 2.0` 으로 표기하여 **부동소수점 연산**을 명시적으로 보장
  - 만약 `1 / 2`로 쓰면 정수 나눗셈으로 결과가 `0`이 되어 오류 발생
- 반환형이 `double`이므로 소수점 결과(7.5) 정상 출력

### 4. Setter 함수 `setWidth()`, `setHeight()`

```cpp
void setWidth(int a)  { width = a; }
void setHeight(int b) { height = b; }
```

- `private` 멤버 변수인 `width`, `height`를 **외부에서 간접적으로 수정**하기 위한 함수
- `setWidth(3)` 호출 시 `width`가 3으로, `setHeight(5)` 호출 시 `height`가 5로 변경됨
- **캡슐화 원칙**을 유지하면서 데이터를 안전하게 변경하는 역할

---

## 자동 인라인 함수

모든 멤버 함수가 **클래스 선언부 내부**에 몸체까지 작성되어 있으므로, 별도의 `inline` 키워드 없이도 컴파일러가 자동으로 인라인 함수로 처리한다.

| 처리 방식 | 소스코드 | 실행 파일(기계어) |
|-----------|----------|-------------------|
| 일반 함수 | 함수 존재 | 함수 존재 (호출 오버헤드 발생) |
| **자동 인라인 함수** | **함수 존재** | **함수 없음 (코드 직접 삽입)** |

→ 소스코드에서는 함수로 작성해 OOP 특성을 유지하고, 실행 시에는 함수 호출 오버헤드 없이 빠르게 동작한다.

---

## main() 함수 실행 흐름

```
① Triangle tri;
   └─ 기본 생성자 자동 호출
      width=1, height=1 초기화
      출력: "폭1, 높이1 인 삼각형 생성"

② tri.setWidth(3);
   └─ width = 3 으로 변경

③ tri.setHeight(5);
   └─ height = 5 로 변경

④ tri.getArea() 호출
   └─ (1.0/2.0) * 3 * 5 = 7.5 반환
      출력: "삼각형의 면적은 7.5"

⑤ return 0; (main 종료)
   └─ 소멸자 자동 호출
      출력: "폭3, 높이5 인 삼각형 소멸"
```

---

## 실행 결과

```
폭1, 높이1 인 삼각형 생성
삼각형의 면적은 7.5
폭3, 높이5 인 삼각형 소멸
```
